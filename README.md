# config-manager

## Installation
1. git clone https://github.com/Da1ne/.dotfiles.git inside $HOME directory
2. cd .dotfiles
3. stow .
Files are now symlinked.
Changes in symlinks will reflect in upstream .dotfiles folders
Git push to sync changes
